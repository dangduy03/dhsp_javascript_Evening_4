Hướng dẫn chạy 
Mở 2 terminal 
Terminal 1 
- Frontend
Vào frontend: Cd frontend
Tải thư viện npm i 
Chạy frontend :  Npm strat
Tải thư viện node-modules : npm install
 
Terminal 2
- Backend 
vào Backend: cd backend
Tải thư viện : npm init 
Thư viện hỗ trợ : npm i express mongodb mongoose dotenv cors cookie-parser jsonwebtoken bcryptjs
Chạy backend: npm run start-dev
